# createCreditLine

This method creates a credit line for an order. It sends a request to the
[Create Credit Line](https://docs.medusajs.com/api/admin#orders\_postordersidcredit-lines) API route.

## Example

```ts
sdk.admin.order.createCreditLine(
  "order_123",
  {
    amount: 100,
    reference: "order",
    reference_id: "order_123",
  }
)
.then(({ order }) => {
  console.log(order)
})
```
