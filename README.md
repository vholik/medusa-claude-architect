# Medusa Architect Claude Plugin

A Claude Code plugin for building Medusa.js e-commerce applications with expert guidance, best practices, and up-to-date documentation.

## Quick Install

```bash
# Add the marketplace
/plugin marketplace add vholik/medusa-claude-architect

# Install the plugin
/plugin install medusa-architect@medusa-architect

# Fetch the latest Medusa.js codebase for context-aware assistance
/medusa-architect:install
```

## Available Skills

### Medusa Docs

Expert guidance for building Medusa e-commerce applications with best practices for architecture, API patterns, and data modeling. Ensures adherence to Medusa 2.0+ conventions and optimal performance patterns.

**Triggers automatically when:**

- Working with Medusa.js codebases
- Asking questions about Medusa architecture
- Implementing e-commerce features
- Debugging Medusa-related issues
- Setting up API routes and workflows

**What it provides:**

- Accurate, up-to-date information from the Medusa codebase
- Best practices for Medusa 2.0+ architecture
- API pattern recommendations
- Data modeling guidance
- Concise, practical code examples with explanations

## Use Cases
